# WizFood

WizFood-react is the Frontend of WizFood, which is my Capstone project to graduate

## Set up
Please make sure you have installed `node`, `npm`, and `yarn`
```bash
$ brew install node
$ brew install npm
$ brew install yarn
```

Install the project
```bash
npm install
```
Run the project locally 
```bash
npm start
```
or 
```bash
yarn start
```
